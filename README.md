# STM32L432KC
STM32L432KC with LIGHT ON

## 보고서용 프로젝트 진행 현황

본 프로젝트는 임베디드 제어 플랫폼을 기반으로 센서 입력, 상태 판단, 외부 장치 연동을 포함하는 제어 시스템의 기본 구조를 구축하는 것을 목표로 한다. 현재까지 개발 환경 구성, 기본 입출력 동작 확인, 통신 기반 구현, 핵심 판단 로직의 초기 구현을 완료하였다.

현재 단계는 실제 하드웨어를 모두 연결하여 검증한 완성 단계가 아니라, 하드웨어 연동을 위한 소프트웨어 기반을 마련한 단계이다. 따라서 보고서에는 현재 성과를 "소프트웨어 기반 구현 완료 및 하드웨어 연동 검증 예정"으로 정리하는 것이 적절하다.

## 진행 내용 요약

- 임베디드 보드 기반 개발 환경 구성
- 기본 구동 흐름 및 입출력 구조 구현
- 외부 장치와의 연동을 위한 통신 기반 구축
- 입력 신호의 신뢰성 판단을 위한 소프트웨어 로직 구현
- 시스템 통합을 위한 모듈 구조 정리

## 향후 진행 계획

- 실제 센서 입력을 반영한 보정 및 동작 확인
- 하드웨어 장치와의 단계별 연동 시험
- 통합 제어 흐름 적용
- 안전 조건 및 예외 상황 검증
- 최종 시스템 동작 테스트

## 보고서 작성 시 표현 방향

보고서에는 구체적인 회로 구성, 통신 세부값, 내부 로직 조건, 장치별 명령 구조 등은 직접적으로 작성하지 않는다. 대신 개발 단계와 성과를 중심으로 추상적으로 표현한다.

권장 표현:

- 제어 시스템의 소프트웨어 기반 구축
- 입력 신호 판단 구조 구현
- 외부 장치 연동을 위한 통신 기반 마련
- 하드웨어 연동 및 통합 검증 예정
